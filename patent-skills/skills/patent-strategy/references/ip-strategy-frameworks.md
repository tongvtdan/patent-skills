# IP Strategy Frameworks

## 1. The Thiel Monopoly Applied to Patents

Peter Thiel argues every great business creates a monopoly. Patents are the most direct way to create a **legal monopoly** in technology.

**The patent moat model:**
```
Core invention patent (broadest independent claim)
    ↓
Improvement patents (continuation applications)
    ↓
Application patents (specific use cases)
    ↓
Manufacturing process patents
    ↓
Design patents (aesthetic protection)
```

Each layer makes it harder for a competitor to replicate your technology without infringing something.

**Key insight**: One strong patent is better than ten weak ones. Invest in quality before quantity.

## 2. The Patent Fence Strategy

Instead of a single "wall" patent, build a fence of overlapping patents:
- Different claim types (product + method + system)
- Different embodiments of the same invention
- Upstream and downstream applications
- Core technology + enabling technology + peripheral features

A competitor must clear ALL fences, not just one.

## 3. Offensive vs. Defensive IP Posture

| Posture | When to Use | Strategy |
|---------|------------|---------|
| **Offensive** | Market leader with clear technical differentiation | Assert patents aggressively; license for revenue |
| **Defensive** | Operating in crowded patent space | Build portfolio to deter suits; cross-license |
| **Balanced** | Growth-stage startup | Core offensive patents + defensive coverage |
| **Open** | Platform / ecosystem play | Open source some IP; patent only what matters |

## 4. The Startup IP Timeline

```
Pre-Seed:
  → File provisional on core invention ($300-500 USD equivalent)
  → Document invention with date-stamped lab notebooks

Seed:
  → Convert provisional to PCT or national filing
  → File on 2-3 key additional inventions
  → Ensure assignment agreements with all founders and employees

Series A:
  → 3-5 granted or pending patents expected by investors
  → Freedom-to-operate analysis in main market
  → IP ownership clearly in company (not individuals)

Series B+:
  → Systematic IP capture process
  → Regular IP committee meetings
  → Active enforcement policy
  → Licensing revenue target
```

## 5. IP Due Diligence Checklist (Investor Perspective)

Investors check these items. Prepare for all of them:

**Ownership**
- [ ] All inventors named correctly on each application
- [ ] Assignment agreements signed by all inventors (especially co-founders who left)
- [ ] Employee IP agreements in place for all engineers
- [ ] No prior employer IP contamination (founders from large tech companies)

**Portfolio Quality**
- [ ] At least one granted patent (not just pending)
- [ ] Claims cover the actual product — not just related technology
- [ ] Independent claims are broad enough to be meaningful
- [ ] No obvious invalidity risks (strong prior art not addressed)

**Status and Maintenance**
- [ ] All maintenance fees paid and current
- [ ] Prosecution timelines tracked — no missed deadlines
- [ ] International filings in key markets (at minimum: home country + US)

**FTO**
- [ ] No blocking third-party patents in primary market
- [ ] FTO analysis documented

**Red Flags for Investors**
- Inventors listed as individuals, not company — means company may not own the patents
- Key patents still pending after 5+ years — may never grant
- Only design patents, no utility patents
- Claims that only cover preferred embodiment, not the concept

## 6. Vietnam-Specific Strategic Considerations

### State-Owned Research IP
If your invention came from state-funded research (university grants, government programs):
- Decree 76/2018/NĐ-CP governs commercialization
- You may need government approval for exclusive licensing
- Consider spin-out structure to cleanly assign IP

### Vietnamese Inventor Identity
In Vietnamese companies, founders often list themselves as inventors but assign to the company. Make sure:
- Invention assignment clauses in employment/founder agreements
- NOIP records show company as applicant/assignee
- Inventors sign inventor declarations correctly

### SHTP and Hi-Tech Zone Benefits
If operating from SHTP (Saigon Hi-Tech Park) or Hoa Lac:
- Apply for hi-tech enterprise certification → reduced IP-related taxes
- Access to SHTP's IP support programs
- Eligibility for government IP development grants
