# Vietnam NOIP Patent Application Format & Filing Checklist

## Required Documents (Article 100 IP Law)

| Document | Vietnamese Name | Required? | Notes |
|----------|----------------|-----------|-------|
| Request form (SC-01) | Tờ khai | Mandatory | Official form from NOIP website |
| Description | Bản mô tả sáng chế | Mandatory | Full specification |
| Claims | Yêu cầu bảo hộ | Mandatory | Must be in Vietnamese |
| Abstract | Tóm tắt | Mandatory | Max 150 words |
| Drawings | Bản vẽ | If referenced in description | Black and white line drawings |
| Priority document | Tài liệu ưu tiên | If claiming Paris Convention priority | Certified copy from original office |
| Assignment document | Tài liệu chuyển nhượng | If applicant ≠ inventor | Signed, notarized |
| Power of Attorney | Giấy ủy quyền | If filed through agent | Original signature |
| Filing fee receipt | Biên lai nộp phí | Mandatory | Pay at NOIP counter or by bank transfer |

## Formatting Requirements

**Page Setup:**
- A4 paper (210mm × 297mm)
- Margins: Left 25mm, Right 20mm, Top 20mm, Bottom 20mm
- Font: Times New Roman or Arial, 12pt (or equivalent)
- Line spacing: 1.5 for description; single space for claims
- Page numbering: Bottom center, starting from page 1

**Description Structure (Vietnamese order):**
1. Tên sáng chế (Title)
2. Lĩnh vực kỹ thuật (Technical Field)
3. Tình trạng kỹ thuật (Background / State of the Art)
4. Bản chất của sáng chế (Nature / Summary of Invention)
5. Mô tả chi tiết các ví dụ thực hiện sáng chế (Detailed Description)
6. Ví dụ (Examples — if applicable)

**Claims (Yêu cầu bảo hộ):**
- Separate document or clearly separated section
- Independent claims first, numbered consecutively
- Each claim: single paragraph, ends with period
- Dependent claims clearly reference parent claim number

**Abstract (Tóm tắt):**
- Maximum 150 words
- Separate page
- No claims, no drawings referenced

## Key Prosecution Timeline

| Milestone | When | Action Required |
|-----------|------|-----------------|
| Filing date | Day 0 | Priority date locked |
| Formality check | 1 month | Fix any formal deficiencies |
| Publication | 18 months from filing | Application published in NOIP Official Gazette |
| Examination request deadline | **42 months** from filing | MUST FILE — or application lapses |
| Substantive examination | 12–24 months after request | NOIP examines claims |
| Office action response | Within time set by NOIP (typically 2-3 months) | Respond or request extension |
| Grant | After examination passed | Certificate issued |
| Annual fees | Annually from grant date | Pay to maintain patent in force |

## Common Formal Deficiencies (Avoid These)

- Missing inventor signatures on declaration
- Power of attorney not notarized (required for foreign applicants)
- Claims not in Vietnamese
- Priority document not certified
- Drawings not black and white (no color)
- Abstract exceeds 150 words
- Reference numerals in description don't match drawings
- Filing fee insufficient (count all independent claims)
