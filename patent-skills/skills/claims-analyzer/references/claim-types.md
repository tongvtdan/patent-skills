# Patent Claim Types Reference

## The Main Claim Categories

### 1. Product Claims
**What**: Claims to a physical thing — device, apparatus, composition, system

**Formats**:
- "A device comprising..."
- "An apparatus for [purpose] comprising..."
- "A system including..."
- "A composition comprising..."

**Best for**: Hardware, physical products, materials, chemical compounds

**Scope tip**: Product claims are infringed by making, using, selling, or importing the product — even if the infringing party uses a different method to make it.

---

### 2. Method / Process Claims
**What**: Claims to a series of steps performed in sequence

**Format**:
```
A method for [achieving result], comprising:
  [verb]-ing [element];
  [verb]-ing [element]; and
  [verb]-ing to produce [output].
```

**Best for**: Manufacturing processes, software methods, treatment methods, biological processes

**Scope tip**: Method claims are infringed only when all steps are performed (usually by one party). Useful when you can't claim the end product but CAN claim the way it's made/used.

---

### 3. System Claims
**What**: A combination of elements working together as an integrated whole

**Format**:
```
A system for [purpose], comprising:
  a [first component] configured to [function];
  a [second component] in communication with the first component; and
  a [processor/controller] configured to [coordinate].
```

**Best for**: Software-hardware combinations, networks, integrated technical systems

---

### 4. Composition Claims
**What**: Claims to a mixture, formulation, or chemical composition

**Format**:
```
A composition comprising:
  from 10% to 40% by weight of [component A];
  from 5% to 20% by weight of [component B]; and
  a carrier.
```

**Best for**: Pharmaceutical formulations, food products, materials, chemical mixtures

---

### 5. Use Claims
**What**: Claims to the use of a known substance for a new purpose

**Format**:
- "Use of [compound X] for treating [disease Y]"
- "Use of [composition] in the manufacture of [medicament] for [treatment]"

**Best for**: Second medical use patents; repurposing of known compounds
**Note**: Not accepted in all jurisdictions (stronger in EPO than USPTO)

---

### 6. Markush Claims (Genus Claims)
**What**: Claims to a group of alternatives defined by common structural or functional features

**Format**:
```
A compound of the formula:
  R-X-Y-Z
  wherein R is selected from the group consisting of [A], [B], and [C];
  X is [chosen from list]; and
  Y is [chosen from list].
```

**Best for**: Chemistry, pharmaceutical chemistry, materials with structural variations

---

### 7. Means-Plus-Function Claims (§112(f) / Functional Claims)
**What**: Claims a function without specifying the exact structure ("means for [doing X]")

**Format**:
```
A device comprising:
  means for detecting [phenomenon]; and
  means for transmitting [signal].
```

**Important**: In USPTO practice, "means for" claims are construed NARROWLY — only to structures disclosed in the specification. Use with caution. Avoid if possible unless you have a very specific structure you want to capture.

**Better alternative**: "a detector configured to detect [phenomenon]" — functional language without means-plus-function interpretation.

## Dependent Claim Strategies

### The Fallback Ladder
Structure dependent claims as progressively narrower fallbacks:

```
Claim 1 (broadest — independent): The core inventive concept
Claim 2 (narrows element A): Adds preferred material/structure for A
Claim 3 (narrows element B): Adds preferred configuration for B
Claim 4 (adds element C): Adds optional but valuable feature
Claim 5 (fully preferred embodiment): Everything combined
```

If claim 1 is invalidated, claims 2-5 survive. Patent owner still has protection on the preferred embodiment.

### Method + Product Pair
Always consider writing BOTH:
- Product claim: "A widget comprising..."
- Method claim: "A method of using a widget, comprising..."

This catches infringers who use but don't make (method claim), and those who make but don't use (product claim).

### Swiss-Type Medical Use
For medical applications, consider:
- Vietnam/EPO: "Use of [X] for the treatment of [Y]"
- US equivalent: Method claim ("A method of treating Y comprising administering X")

## Common Claim Drafting Vocabulary

| Term | Meaning | Scope Implication |
|------|---------|------------------|
| comprising | Including (open) | Other elements OK; broad |
| consisting of | Only these elements | Very narrow; rare |
| consisting essentially of | These + non-material elements | Medium; for materials |
| wherein | Introduces a limitation on a preceding element | Narrowing |
| further comprising | Adds an additional element | Narrowing by addition |
| configured to | Functional language for structure | Standard; acceptable |
| adapted to | Designed/modified for function | Slight narrowing vs. "configured to" |
| operably connected | Connected in a way that allows function | Relationship claim |
| in communication with | Signal/data communication link | Electronic/digital claims |
| a/an | First introduction of element | Open; not limiting |
| the | Reference back to introduced element | Requires antecedent |
