# /build-ip-strategy — IP Strategy & Portfolio Roadmap

Build a strategic IP plan that turns inventions into competitive advantage. Covers IP audit, filing roadmap, budget, competitive landscape, and investor-ready IP positioning.

## Invocation

```
/patent-skills:build-ip-strategy
/patent-skills:build-ip-strategy startup raising Series A in 6 months
/patent-skills:build-ip-strategy licensing  (→ IP monetization and licensing strategy)
/patent-skills:build-ip-strategy fundraising  (→ VC due diligence ready IP plan)
```

## Workflow

Apply the `patent-strategy` skill:

### 1. IP Audit
Inventory all current and pipeline IP assets — patents, trade secrets, designs, trademarks.

### 2. Patentability Triage
For each invention: file / don't file / publish defensively — with rationale.

### 3. Jurisdiction Filing Roadmap
| Invention | Jurisdiction | Filing Type | Timeline | Est. Cost |
|-----------|-------------|-------------|----------|-----------|
Map where to file, when, and why.

### 4. Budget Plan
3-year IP investment plan in VND and USD.

### 5. Competitive IP Landscape
Key competitor patents, design-around opportunities, FTO risks.

### 6. 12-Month Action Plan
Prioritized month-by-month actions with owners and deadlines.

## Strategic Modes

**`/build-ip-strategy startup`** — Early-stage focus: minimum viable IP to protect core invention + satisfy seed investors

**`/build-ip-strategy fundraising`** — VC-ready IP package: portfolio summary, ownership verification, FTO status, valuation memo

**`/build-ip-strategy licensing`** — Revenue-focused: identify licensable IP, royalty rate benchmarks, target licensees, deal structure

## Output
IP Strategy document saved as `ip-strategy-[company]-[date].docx`
